# Security Policy

## Supported Version

Lockrion Issuance Contract v1.1 is the current supported version.

Only the latest tagged release is considered supported.

---

## Reporting a Vulnerability

If you discover a security vulnerability, do NOT open a public issue.

Please report responsibly via:

Email: support@lockrion.com  
Subject: [SECURITY] Lockrion v1.1 Vulnerability Report

Include:

- Description of the issue
- Reproduction steps
- Affected version
- Potential impact
- Proof-of-concept (if available)

---

## Scope

Security reports may include:

- Arithmetic overflow/underflow
- Escrow integrity violations
- PDA validation bypass
- Determinism violations
- Settlement bypass
- Authority misuse
- Reentrancy-like behavior via CPI
- Seed mutation vulnerabilities

Out of scope:

- UI issues
- Documentation typos
- Feature requests

---

## Response Process

- Acknowledgement within 72 hours
- Impact assessment
- Fix or mitigation plan
- Coordinated disclosure after patch

Responsible disclosure is required.
Public exploitation attempts will be considered hostile.
